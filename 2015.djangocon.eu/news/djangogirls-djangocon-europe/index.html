Django Girls at DjangoCon Europe

Django Girls is coming to Cardiff for the first time - local volunteers and helpers are needed.

We’re really thrilled that there’ll be an all-day Django Girls workshop at DjangoCon this year.

Django Girls is a superb initiative that has successfully introduced hundreds of women to Python programming and web development in Django.

The workshop is absolutely free.

Everything you need to know is on the event page - don’t miss it!
