The Cardiff - Bristol Airport bus link has been discontinued

Travellers coming to Cardiff via Bristol Airport should note that the previously-recommended Greyhound Bus link will no longer be available

Please see Getting here and in particular the Bristol Airport travel information if you had been planning to use the bus.

The alternatives now available are a little less convenient and more expensive, but should have little effect on journey times.

Sorry for the inconvenience.
