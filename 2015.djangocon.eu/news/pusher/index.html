Pusher, our Silver sponsor

Build powerful realtime features into your applications with Pusher.

Pusher is a real-time evented messaging platform with hosted APIs and open-source libraries that greatly simplify integrating real-time functionality into web and mobile applications.

Features such as in-app notifications, activity streams, chat, real-time dashboards and multi-user collaboration apps can be added with just a few lines of code with the language and technology of your choice; from back-ends built with any Python framework - such as Django - to front-ends in vanilla JS, Angular, Backbone, Ember, Knockout, React or native Android and iOS.

As a hosted service, Pusher will automatically scale when required, removing all the pain of setting up and maintaining a secure, real-time infrastructure.

Pusher is already trusted to do so by thousands of developers and companies like GitHub, MailChimp, the Financial Times, Buffer and many more.
