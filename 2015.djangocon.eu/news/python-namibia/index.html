Python Namibia 2015

Half the DjangoCon Europe team has been in Windhoek, helping run the first-ever Python conference in Namibia.

If we’ve been a bit quiet this week or slow responding to email enquiries, it’s because four of the eight organising committee members have been exceptionally busy with another Python event: Python Namibia 2015.

That’s over now, and we’ll soon all be back in Cardiff, not only exhausted but also very happy with how it went, and full of ideas to be incorporated into future events.

The event was well-supported by the international Python/Django communities, and you’ll be able to hear more about it at DjangoCon in June - and, we hope, to meet some Namibian DjangoNauts, if we can make it possible for them to be here.
