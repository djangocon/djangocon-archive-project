Django for web designers

A workshop aimed at web designers who want to take the next step and become programmers.

Note: this session is part of DjangoCon Europe’s open day. It’s free and all are welcome to attend, but please register because places are limited.

About the workshop

This workshop will walk students through the process of building a web app with Django - from conception to deployment. It’s aimed at completely new programmers, especially those who aren’t comfortable with the command line.

In the workshop we work to the strengths and skills of web designers, and start with the templates and webpages that they’re more familiar with, and quickly proceeding to results in the browser from their newly-created live web application.

Students will get free e-copies of Tracy’s new book to use during and after the workshop.

Please bring your laptop. It’ll be good to already have Python installed, but we can help if you don’t.

About Tracy Osborn

Tracy Osborn is a designer who became a web application programmer. In 2011 she decided to learn Python and Django, and six weeks later she launched her first web application.

Since then she has built a successful Django-powered business and written a book, Hello Web App, an introduction to web programming with Django.
