Building a usable & beautiful Admin

Make the most of the many useful and not-so-known features of Django's Admin to improve the experience of your users.

Note: this session will run alongside the conference sprints. It’s free to all conference attendees.

About the workshop

The Django Admin is a great feature that allows us to create a beautiful Admin interface in a matter of minutes. However, while building an admin interface, it’s easy to forego basic UX rules.

In this workshop, Yigit will demonstrate some key points to make the Admin interface more about users. Starting from an existing (ready-to-use) project, we will create the default Admin page, customise it by using ModelAdmin features, and finish up by adding a nice Bootstrap skin: Django Suit.

About Yiğit Güler

Yiğit is a backend development lead at Hipo, and is heavily involved in Instanbul’s Django community.
