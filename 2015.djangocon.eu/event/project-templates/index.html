Build your own project template

A hands-on workshop, introducing why and how you can use CookieCutter to make starting new Django projects faster, richer and better

Note: this session will run alongside the conference sprints. It’s free to all conference attendees.

About the workshop

Although djangoadmin startproject —template is great, Cookiecutter allows for even more automation, which is better still, since we all want to cut down on boilerplate code.

Cookiecutter is therefore the logical next step in custom project template creation.

In this workshop, we’ll first design a project template to use with djangoadmin startproject —template, before porting it to Cookiecutter.

Finally we’ll introduce more automation, using CookieCutter hooks to initialise a repository and a virtualenv alongside our new project.

About Emmanuelle Delescolle

Emmanuelle is a full-stack developer who loves Django. She first ran into Python around 1998, and Django in 2009.

She recently started a new Django-centric company, LevIT.

Emmanuelle has been curious about open-source in general ever since she got her first Internet connection and she has also always been passionate about re-sharing what she has been able to learn.
