Introduction to Django REST Framework

RESTful APIs are the future: get started with a hands-on introduction to something that should be in every Django developer's toolbox.

Note: this session will run alongside the conference sprints. It’s free to all conference attendees.

About the workshop

JavaScript frameworks, external tools and service-oriented architectures have made APIs an essential part of modern web projects.

Although it is possible to build those APIs from scratch with Django, using an existing REST framework is an easier, faster and more secure way to get there.

After a short introduction to REST APIs, we will focus on building one, using Django REST framework.

Various components of the framework will be introduced (permissions, filtering, human-usable interface) to get a fully-featured API including some tests.

A basic knowledge of Python and Django is expected.

About Xavier Ordoquy

Xavier is a software developer, problem solver and long-time freelancer.

He loves to help compagnies adopt best practices or train teams in Python/Django.
